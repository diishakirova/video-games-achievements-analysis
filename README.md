# Video Games Achievements Analysis (Superset)

BI-проект по анализу игровых достижений, жанров и платформ с использованием Apache Superset.

## Ссылка на дашборд
https://superset.bi-analysts.education-services.ru/superset/dashboard/p/vqGqpzO58lg/

## Цель проекта
Проанализировать распределение игровых достижений:
- по жанрам
- по платформам
- по играм
- долю редких достижений

## Основные метрики
- Общее количество достижений
- Процент редких достижений по жанрам
- Количество достижений в каждой игре
- Количество достижений по жанрам и платформам

## Используемые инструменты
- SQL
- Apache Superset
- Data visualization
- BI-аналитика

## Бизнес-инсайт
Проект позволяет определить:
- жанры с наибольшей концентрацией достижений
- платформы с максимальной вовлеченностью игроков
- распределение редких достижений
